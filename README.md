# AlineacionBot
Bot for Telegram written in Nodejs

This Bot allows the user to create football line-ups.

**How to use the bot:**
* Start conversation in Telegram with : `@AlineacionBot` 
* Select desired lineup
* Introduce the name of the players in order
* Get your image!
